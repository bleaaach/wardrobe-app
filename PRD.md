# 智能衣橱 PRD

## 1. 产品概述

**产品名称**：智能衣橱 (Wardrobe App)

**一句话描述**：拍照管理衣物、自由搭配组合、每日穿搭记录的移动端衣橱管理工具。

**核心价值**：把衣柜数字化，让用户一眼看到自己有哪些衣服、怎么搭配、每天穿了什么。

**平台**：Android APK + Web 网页版（React Native / Expo 跨平台）

---

## 2. 目标用户

- 想要管理自己衣物的人（尤其女性用户）
- 经常忘记自己有什么衣服、重复购买的人
- 喜欢搭配、记录每日穿搭的人
- 需要多设备同步数据的人（手机 + 电脑）

---

## 3. 功能模块

### 3.1 衣橱管理

| 功能 | 描述 | 状态 |
|------|------|------|
| 添加衣物 | 拍照/相册选图，选分类，填名称，标记季节 | ✅ |
| 分类浏览 | 按上衣/裤子/裙子/外套/鞋子/配饰/包包两级筛选 | ✅ |
| 衣物详情 | 大图查看、编辑（名称/分类/品牌/颜色/季节/尺码/价格/存放位置/标签/备注）、收藏、删除 | ✅ |
| 搜索 | 按名称搜索 | ✅ |
| 抠图去背景 | 添加衣物时一键去除照片背景（调用服务端 Python 脚本） | ✅ |
| 批量导入 | 从 `.zip` 备份文件导入衣橱数据（Web 端） | ✅ |
| 软删除与恢复 | 删除衣物进入回收站，支持恢复 | ✅ |

### 3.2 搭配管理

| 功能 | 描述 | 状态 |
|------|------|------|
| 创建搭配 | 从衣橱多选衣物组合成一套搭配，支持直接保存或拼图编辑 | ✅ |
| 搭配列表 | 浏览所有搭配，展示缩略图与元信息 | ✅ |
| 搭配详情 | 查看搭配包含的衣物，展示拼图效果，支持删除 | ✅ |
| 拼图编辑 | 自由拖拽调整衣物在搭配图中的位置与层级 | ✅ |
| 随机搭配 | 一键随机生成搭配 | 🔲 规划中 |
| AI 搭配推荐 | 根据天气/场合推荐搭配 | 🔲 远期 |

### 3.3 穿搭日历

| 功能 | 描述 | 状态 |
|------|------|------|
| 月历视图 | 查看每日穿搭记录，有记录的日期显示衣物缩略图 | ✅ |
| 添加记录 | 选择日期，从搭配库/单品库/相册照片/复制昨天四种方式记录 | ✅ |
| 拍照记录 | 拍摄或上传实际穿搭效果照片 | ✅ |
| 统计回顾 | 最常穿的衣物统计（在统计页展示） | ✅ |
| 穿着次数 | 记录搭配时自动关联更新衣物的 `wearCount` | ✅ |

| 回收站 | 查看已删除衣物，支持恢复 | ✅ |

| 云服务器同步 | 数据同步到自有服务器 (8.162.26.192) | ✅ 后端已部署 |

## 4. 技术架构

```
┌─────────────────────────────────────┐
│          React Native / Expo         │
│  ┌─────────┐  ┌─────────┐  ┌──────┐ │
│  │ Android │  │   Web   │  │ iOS  │ │
│  │  APK    │  │  PWA    │  │ (未来)│ │
│  └────┬────┘  └────┬────┘  └──────┘ │
│       │            │                 │
│  ┌────┴────────────┴────┐            │
│  │   Expo Router +       │           │
│  │   Zustand State       │           │
│  └─────────┬─────────────┘           │
│            │                          │
│  ┌─────────┴─────────────┐           │
│  │   localStorage (本地)  │           │
│  │   IndexedDB (图片 Blob)│           │
│  └─────────┬─────────────┘           │
│            │ REST API                 │
│  ┌─────────┴─────────────┐           │
│  │   Express Sync Server  │           │
│  │   (8.162.26.192:3001) │           │
│  │   SQLite 数据库         │           │
│  └───────────────────────┘           │
└─────────────────────────────────────┘
```

**技术栈**：
- 前端：React Native 0.85 + Expo SDK 56 + Expo Router
- 状态管理：Zustand
- 本地存储：localStorage（JSON 数据）+ IndexedDB（图片 Blob）
- 样式：NativeWind + Tailwind CSS v4
- UI/交互：@expo/vector-icons、react-native-reanimated、react-native-gesture-handler
- 原生能力：expo-image-picker、expo-file-system、expo-sharing、expo-sqlite（已引入，待迁移）
- 后端：Node.js + Express + SQLite
- 部署：Nginx 反向代理，Alibaba Cloud ECS
