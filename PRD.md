# 智能衣橱 PRD

## 1. 产品概述

**产品名称**：智能衣橱 (Wardrobe App)

**一句话描述**：拍照管理衣物、自由搭配组合、每日穿搭记录的移动端衣橱管理工具。

**核心价值**：把衣柜数字化，让用户一眼看到自己有哪些衣服、怎么搭配、每天穿了什么。

**平台**：Android APK + Web 网页版（React Native / Expo 跨平台）

---

## 2. 目标用户

- 想要管理自己衣物的人（尤其女性用户）
- 经常忘记自己有什么衣服、重复购买的人
- 喜欢搭配、记录每日穿搭的人
- 需要多设备同步数据的人（手机 + 电脑）

## 4. 技术架构

```
┌─────────────────────────────────────┐
│          React Native / Expo         │
│  ┌─────────┐  ┌─────────┐  ┌──────┐ │
│  │ Android │  │   Web   │  │ iOS  │ │
│  │  APK    │  │  PWA    │  │ (未来)│ │
│  └────┬────┘  └────┬────┘  └──────┘ │
│       │            │                 │
│  ┌────┴────────────┴────┐            │
│  │   Expo Router +       │           │
│  │   Zustand State       │           │
│  └─────────┬─────────────┘           │
│            │                          │
│  ┌─────────┴─────────────┐           │
│  │   AsyncStorage (本地)  │           │
│  └─────────┬─────────────┘           │
│            │ REST API                 │
│  ┌─────────┴─────────────┐           │
│  │   Express Sync Server  │           │
│  │   (8.162.26.192:3001) │           │
│  │   SQLite 数据库         │           │
│  └───────────────────────┘           │
└─────────────────────────────────────┘
```

**技术栈**：
- 前端：React Native 0.85 + Expo SDK 56 + Expo Router
- 状态管理：Zustand
- 本地存储：AsyncStorage
- UI 设计：自定义设计系统（暖色米色调）
- 后端：Node.js + Express + SQLite
- 部署：Nginx 反向代理，Alibaba Cloud ECS
